# Документация MIO Kitchen

Это дерево описывает текущее Python-приложение на русском языке. Источником истины является production-код в `src`; проверки документации сверяют структуру, ссылки, команды и выбранные архитектурные утверждения с репозиторием.

```mermaid
flowchart LR
  START["Выберите задачу"] --> ARCH["Разобраться в устройстве системы"]
  START --> CHANGE["Внести изменение"]
  START --> VERIFY["Проверить или собрать релиз"]
  ARCH --> MAP["Полная карта архитектуры"]
  CHANGE --> BOUNDS["Границы модулей и runtime-окружение"]
  VERIFY --> CHECKS["Тесты и скрипты"]
  CHECKS --> BUILD["Windows-сборка и релиз"]
```
